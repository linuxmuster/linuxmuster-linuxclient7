About¶

Welcome in this API reference documentation !

Linuxmuster-linuxclient7 is a part of the linuxmuster.net’s project which provides a complete environment for school network management.

You can find the whole description of the install process on the official documentation page of the project.

Particularly, some appliances (Proxmox, XCP-ng and KVM) are prepared in order to permit a fast deployment for test use.
